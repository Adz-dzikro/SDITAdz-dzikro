Fasilitas & Program Unggulan Sekolah

Menyediakan Fasilitas Terbaik dan Program Unggulan untuk Pendidikan Berkualitas

Fasilitas Sekolah

Gambar Perpustakaan

Perpustakaan

Perpustakaan modern dengan koleksi buku yang lengkap, ruang baca yang nyaman, dan akses internet untuk mendukung kegiatan pembelajaran siswa. Dilengkapi dengan sistem katalog digital dan area diskusi kelompok.

Gambar Laboratorium

Laboratorium Sains

Laboratorium sains lengkap untuk praktikum Fisika, Kimia, dan Biologi. Dilengkapi dengan peralatan modern dan sistem keamanan standar internasional untuk mendukung pembelajaran praktis siswa.

Gambar Lapangan Olahraga

Fasilitas Olahraga

Lapangan olahraga multifungsi untuk berbagai aktivitas seperti basket, futsal, dan voli. Dilengkapi dengan ruang ganti dan perlengkapan olahraga yang memadai untuk mendukung kegiatan ekstrakurikuler.

Gambar Lab Komputer

Laboratorium Komputer

Ruang komputer modern dengan perangkat terbaru, koneksi internet cepat, dan software pendidikan untuk mendukung pembelajaran teknologi informasi dan komunikasi.

Program Unggulan

Gambar Program Bilingual

Program Kelas Bilingual

Program pembelajaran dwibahasa (Indonesia-Inggris) untuk mempersiapkan siswa menghadapi persaingan global. Didukung oleh guru berpengalaman dan native speaker.

  • ✨ 90% lulusan meraih skor TOEFL > 550
  • ✨ Kerjasama dengan lembaga bahasa internasional
  • ✨ Program pertukaran pelajar ke luar negeri
Gambar Program Sains

Program Riset Sains

Program khusus untuk pengembangan kemampuan riset dan inovasi di bidang sains. Siswa dibimbing untuk melakukan penelitian dan mengikuti kompetisi sains.

  • ✨ Juara Olimpiade Sains Nasional
  • ✨ Publikasi penelitian siswa di jurnal nasional
  • ✨ Kerjasama dengan perguruan tinggi ternama
Gambar Program Entrepreneur

Program Kewirausahaan

Program pengembangan jiwa entrepreneurship sejak dini. Siswa dibimbing untuk memahami dunia bisnis dan mengembangkan ide kreatif.

  • ✨ Business Plan Competition tingkat nasional
  • ✨ Kerjasama dengan pengusaha sukses
  • ✨ Student Company Program
